V2ray客户端的使用

Posted on 2018-11-18 | In v2ray | | Visitors:

  之前一直使用ss搭梯子,但是最近ss经常被封所以换成了v2ray来搭梯子,v2ray是ss项目作者被约谈以后v2ray项目组表示抗议开发出来的,致力于让大家更好更快的科学上网.本篇文章就介绍一下怎么使用v2ray客户端进行科学上网.

  本篇文章只介绍v2ray客户端的使用,前提条件是你已经有了一个可以进行科学上网的代理服务器v2ray服务端的配置,及连接代理服务器的配置信息.v2ray其实是不分客户端和服务端的,客户端和服务端都是一套程序,只是其配置不一样.

  客户端的使用有两种方式,第一种是直接下载v2ray的程序然后修改其配置文件然后运行就可以了,第二种方式是使用对应平台的第三方客户端程序,其实其核心也是v2ray程序,只是用客户端进行了一层包装让配置更简单.

直接使用v2ray

  首先从v2ray下载对应平台的包,地址:https://github.com/v2ray/v2ray-core/releases,下载完成后解压修改配置文件config.json然后执行v2ray文件就行了.
简单的config.json配置如下:

log:是日志配置
inbounds:本地代理监听的配置,跟ss的配置其实很像,监听本地ip:127.0.01,端口:1080,端口可以更换,
outbounds:出站配置,也就是我们的代理配置,可以设置多个,然后根据路由策略来配置走不通的出站配置.其中address port就是代理服务器的ip地址和端口,users配置与服务端的入站配置保持一致,
routing:路由策略,可以配置那些ip或域名走代理,那些走本地直连,比如这里配置了大陆ip和国内网站域名走直连,其他默认走第一个出战配置也就是我们的代理配置.

配置完以后运行v2ray就可以了,程序就会一直监听本地1080端口,然后将请求通过路由进行转发,在浏览器里或者其他应用配置代理为127.0.0.1:1080及可,跟ss一样.

第三方客户端

除了直接使用v2ray以外还可以使用对应平台的第三方客户端程序.分别如下:

windows:

  • V2RayW: https://github.com/Cenmrev/V2RayW
  • V2RayN: https://github.com/2dust/v2rayN
  • V2RayS: https://github.com/Shinlor/V2RayS
  • V2RayGCon: https://github.com/nobody3u/V2RayGCon

Mac OS X:

  • V2RayX: https://github.com/Cenmrev/V2RayX

iOS:

  • Kitsunebi: 已下架 TestFlight
  • Kitsunebi Lite : iTunes
  • Shadowrocket : iTunes
  • Pepi: iTunes
  • Quantumult:iTunes

iOS的可能需要用美国账号登录才能下载

Android:

  • BifrostV: Play Store APK Pure
  • V2RayNG: Play Store GitHub
  • V2Ray Go: Play Store Github
  • Actinium: Play Store (已下架) Github
